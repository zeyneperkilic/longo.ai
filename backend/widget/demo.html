🏥 Longo Health Widget Demo

📱 Widget Nasıl Görünür?

Sağ alt köşede mavi yuvarlak bir buton göreceksiniz. Bu butona tıkladığınızda chat penceresi açılacak.

Widget Özellikleri:

  • 🔵 Mavi yuvarlak chat butonu (Kahve.com stilinde)
  • 💬 Şeffaf glassmorphism chat arayüzü
  • 🖼️ Longo fotoğrafı ve animasyonları
  • 📝 Gelişmiş typing efektleri
  • 🎨 Modern ve şeffaf tasarım

🔧 Entegrasyon Kodu

Widget'ı kendi sitenize entegre etmek için bu kodu ekleyin:

<!-- CSS dosyasını ekleyin --> <link rel="stylesheet" href="longo-health-widget.css"> <!-- JavaScript dosyasını ekleyin --> <script src="longo-health-widget.js"></script>

📋 Widget Dosyaları

Widget için gerekli dosyalar:

  • longo-health-widget.js - Ana widget kodu
  • longo-health-widget.css - Widget stilleri
  • demo.html - Bu demo sayfası

🎯 Kullanım

Widget'ı test etmek için:

  1. Sağ alt köşedeki mavi butona tıklayın
  2. Chat penceresi açılacak
  3. Mesaj yazın ve "Gönder" butonuna tıklayın
  4. AI yanıtını bekleyin

💡 Özelleştirme

Widget'ı özelleştirmek için CSS dosyasındaki renkleri ve boyutları değiştirebilirsiniz:

/* Ana renk değişimi */ #chat-button { background: #your-color; /* Buton rengi */ } #longo-chat-header { background: #your-color; /* Header rengi */ }
